Reports string, plural, and array resources defined in Compose Multiplatform XML files that are not referenced anywhere in the project's Kotlin code.

Unused resources increase bundle size and make resource files harder to maintain. This inspection searches the project for references matching the resource's generated accessor (e.g., Res.string.greeting) and highlights resources with no usages found.

Example:


<!-- Highlighted as unused if no Kotlin code references Res.string.old_message -->
<string name="old_message">This message is no longer used</string>

Use the provided 'Remove unused resource' quick fix to delete the resource element from the XML file.
